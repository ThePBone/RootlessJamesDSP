Використовуйте JamesDSP як загальносистемний механізм обробки аудіо, який працює без прав root.

Будь ласка, прочитайте цей документ перед використанням програми, оскільки вона має деякі обмеження, які можуть бути проблемними для деяких користувачів. Для початкового налаштування потрібно Shizuku (Android 11+) або доступ до ADB через комп'ютер.

JamesDSP підтримує такі аудіо ефекти:
* Лімітер (Limiter control)
* Контроль вихідної гучності (Output gain control)
* Автоматичний компресор динамічного діапазону
* Динамічне посилення баса (Dynamic bass boost)
* Інтерполюючий FIR-еквалайзер (Interpolating FIR equalizer)
* Графічний еквалайзер (Arbitrary response equalizer)
* ViPER-DDC
* Конвольвер (Convolver)
* Live-programmable DSP
* Аналогове моделювання (Analog modelling)
* Ширина звукової сцени (Soundstage wideness)
* Кроссфід (Crossfeed)
* Ефект віртуальної кімнати (Reverb)

Крім того, цей додаток безпосередньо інтегрується з AutoEQ. Використовуючи інтеграцію AutoEQ, ви можете шукати та імпортувати частотні характеристики, які спрямовані на корекцію звучання навушників до нейтрального звуку. Для використання перейдіть у розділ 'Графічний еквалайзер > Амплітудна характеристика > Профілі AutoEQ'.

--- Обмеження
* Додатки, які блокують запис внутрішнього аудіо, не будуть оброблені (наприклад, Spotify, Google Chrome)
* Деякі додатки, які використовують апаратне прискорення під час відтворення, можуть спричиняти проблеми та вимагати ручного виключення (наприклад, деякі ігри, створені на платформі Unity)
* Не може бути запущено паралельно з (деякими) іншими додатками для аудіоефектів (наприклад, Wavelet та інші додатки, що використовують Android API `DynamicsProcessing`)


- Роботу додатків підтверджено:
* YouTube
* YouTube Music
* Amazon Music
* Deezer
* Poweramp
* Substreamer
* Twitch
* ...

- Додатки, що не підтримуються:
* Spotify (Примітка: для підтримки Spotify потрібен патч Spotify ReVanced)
* Google Chrome
* SoundCloud
* ...
