Вы можете использовать JamesDSP в качестве общесистемного механизма обработки звука без какого-либо root-доступа.

Пожалуйста, прочтите этот документ перед использованием приложения, так как оно имеет некоторые ограничения, которые могут быть проблемными для некоторых пользователей. Для первоначальной настройки требуется Shizuku (Android 11+) или доступ к ADB через компьютер.

JamesDSP поддерживает следующие аудио эффекты:
* Лимитер (Limiter control)
* Управление выходной громкостью (Output gain control)
* Auto dynamic range compressor
* Динамическая подстройка баса (Dynamic bass boost)
* Интерполирующий FIR-эквалайзер (Interpolating FIR equalizer)
* Arbitrary response equalizer (Graphic EQ)
* ViPER-DDC
* Конвольвер (Convolver)
* Live-programmable DSP (скриптовый движок для аудио эффектов)
* Аналоговое моделирование (Analog modelling)
* Ширина звуковой сцены (Soundstage wideness)
* Кроссфидер (Crossfeed)
* Эффект виртуальной комнаты (Reverb)

Кроме того, это приложение напрямую интегрируется с AutoEQ. Используя интеграцию AutoEQ, вы можете искать и импортировать частотные характеристики, которые направлены на коррекцию звучания наушников до нейтрального звука. Для начала перейдите в раздел 'Arbitrary response equalizer > Magnitude response > AutoEQ profiles'.

--- Ограничения
* Приложения, которые блокируют запись внутреннего аудио, не будут обработаны (например, Spotify, Google Chrome)
* Некоторые приложения, которые используют аппаратное ускорение при воспроизведении, могут вызывать проблемы и требовать ручного исключения (например, некоторые игры, созданные на платформе Unity)
* Не может быть запущено параллельно с (некоторыми) другими приложениями для аудиоэффектов (например, Wavelet и другие приложения, использующие Android API `DynamicsProcessing`)


- Работа приложений подтверждена:
* YouTube
* YouTube Music
* Amazon Music
* Deezer
* Poweramp
* Substreamer
* Twitch
* ...

- Неподдерживаемые приложения:
* Spotify (Примечание: для поддержки Spotify требуется патч Spotify ReVanced)
* Google Chrome
* SoundCloud
* ...
