使用 JamesDSP 作为系统范围的音频处理引擎，无需任何 Root 权限。

请在使用该应用程序之前阅读整篇文档。 初始设置需要通过电脑访问 Shizuku（Android 11+）或 ADB。

JamesDSP 支持以下音效：
* 限制器控制
* 输出增益控制
* 自动动态范围压缩器
* 动态低音增强
* 内置 FIR 均衡器
* 任意响应均衡器（Graphic EQ）
* ViPER-DDC
* 脉冲反馈
* 实时可编程 DSP（音频效果脚本引擎）
* 模拟建模
* 声场广度
* 交叉反馈
* 虚拟场效果（混响）

此外，此应用直接与 AutoEQ 集成。 使用 AutoEQ 集成，您可以搜索并导入旨在将耳机校正为中性声音的频率响应。 转到“任意响应均衡器 > 幅度响应 > AutoEQ 配置文件”开始。

--- 局限性
* 阻止内部音频捕获的应用程序仍未处理（例如 Spotify、Google Chrome）
* 使用某些类型的 HW 加速播放的应用程序可能会导致问题，需要手动排除（例如，某些 Unity 游戏）
* 不能与（某些）其它音频效果应用共存（例如，Wavelet 和其它使用“DynamicsProcessing”Android API 的应用）

- 确认工作的应用：
* YouTube
* YouTube Music
* Amazon Music
* Deezer
* Poweramp
* Substreamer
* Twitch
* ...

- 不支持的应用包括：
* Spotify（注意：Spotify ReVanced 补丁需要支持 Spotify）
* Google Chrome
* SoundCloud
* ...
